# 梱包仕様書（Packaging Specifications）

## 1. 概要

### 梱包仕様書とは

梱包仕様書（Packaging Specifications）とは、製品を顧客に届けるまでの間、**品質を維持したまま安全に保管・輸送する**ための梱包方法・梱包材料・輸送条件を規定した文書です。

製品の物理的な保護だけでなく、輸送中の振動・衝撃・温湿度・静電気などの環境ストレスから製品を守り、顧客が開梱した時点で製品仕様を満たした状態であることを保証します。APQP Phase 3（工程設計・開発）のアウトプットとして、他の工程設計文書と同時期に作成されます。

### 目的

| 目的 | 説明 |
|------|------|
| 製品品質の保持 | 輸送・保管中の振動・衝撃・温湿度変化・静電気・汚染から製品を保護する |
| 輸送中の損傷防止 | 落下・圧縮・振動等の物理的ストレスに耐える梱包構造を設計する |
| 保管条件の規定 | 倉庫・物流センターでの保管方法（段積み数、温湿度条件等）を明確にする |
| 物流効率の最適化 | パレット積載効率・トラック積載効率を最大化し、物流コストを削減する |
| 法規制への適合 | 輸出入規制・環境規制（RoHS、REACH等の梱包材規制）に適合する |
| 開梱作業性 | 顧客・販売店での開梱のしやすさを考慮する |
| 環境配慮 | リサイクル可能な梱包材の使用、梱包材の最小化を推進する |

### 工程設計における位置づけ

```
工程フローチャート（梱包工程を含む全工程の流れを定義）
    │
    ├──→ コントロールプラン（梱包工程の管理項目を規定）
    │
    ├──→ フロアプランレイアウト（梱包エリアの配置計画）
    │
    └──→ 梱包仕様書（梱包方法・材料・輸送条件を規定） ◀ 本記事
              │
              ├──→ 梱包作業指示書（梱包作業の手順を作業者向けに展開）
              │
              ├──→ 梱包材の調達・管理
              │
              └──→ 輸送試験（ISTA、JIS等に基づく梱包品の輸送試験）
```

梱包仕様書は、製品が出荷から顧客到着までの全過程で品質を維持できる梱包を設計するための文書です。PFMEAで特定した輸送中のリスク（振動・落下・温湿度変化等）を梱包構造で対策する役割を担います。

### IATF 16949 / PPAP の要求事項

| 条項・要件 | 内容 |
|-----------|------|
| **IATF 16949 8.5.4** 保存 | 出荷時まで適合性を維持する識別・取扱い・汚染防止・梱包・保管・輸送・保護の適切な処置を取ること |
| **IATF 16949 8.5.4.1** 保存 — 補足 | 保存には識別、取扱い、梱包、保管、保護を含むこと |
| **PPAP 第4版 要素12** | 顧客固有の梱包基準への適合を検証すること。梱包がPPAP提出物の一部であること |
| **PPAP 第4版 要素15** | 部品提出保証書（PSW）で梱包承認を確認すること |

## 2. 梱包の分類

### 梱包の階層

梱包は一般に3つの階層で構成されます。

| 階層 | 名称 | 英語名 | 機能 | 例 |
|------|------|--------|------|-----|
| **一次梱包** | 個装 | Primary Packaging | 製品を直接包装し、傷・汚れ・静電気から保護する | ポリ袋、緩衝材、ESD袋 |
| **二次梱包** | 内装 | Secondary Packaging | 複数の個装品をまとめ、箱に収納する | 段ボール箱、仕切り、トレー |
| **三次梱包** | 外装 | Tertiary Packaging | 輸送単位としてパレットに積載する | パレット、ストレッチフィルム、バンド |

```
┌────────────────────────────────────┐
│ 三次梱包（外装）                      │
│  ┌──────────────────────────────┐ │
│  │ 二次梱包（内装）                  │ │
│  │  ┌────────┐  ┌────────┐  │ │
│  │  │一次梱包  │  │一次梱包  │  │ │
│  │  │（個装）  │  │（個装）  │  │ │
│  │  │ [製品]  │  │ [製品]  │  │ │
│  │  └────────┘  └────────┘  │ │
│  └──────────────────────────────┘ │
│                                      │
│  ┌──────────────────────────────┐ │
│  │ 二次梱包（内装）                  │ │
│  │  ┌────────┐  ┌────────┐  │ │
│  │  │一次梱包  │  │一次梱包  │  │ │
│  │  │（個装）  │  │（個装）  │  │ │
│  │  │ [製品]  │  │ [製品]  │  │ │
│  │  └────────┘  └────────┘  │ │
│  └──────────────────────────────┘ │
│  ═══════パレット═══════════════  │
└────────────────────────────────────┘
```

### 梱包形態の分類

| 形態 | 用途 | 特徴 |
|------|------|------|
| **段ボール梱包** | 一般製品 | 軽量・安価・リサイクル容易。緩衝材と組み合わせて使用 |
| **通い箱（リターナブル）** | 工程間搬送・近距離輸送 | 繰返し使用可能。プラスチック製が多い。環境負荷低減 |
| **木箱梱包** | 重量物・精密機器 | 高い保護性能。輸出時は燻蒸処理（ISPM15）が必要 |
| **真空梱包** | 精密部品・電子部品 | 防湿・防錆。乾燥剤・脱酸素剤と併用 |
| **ESD梱包** | 静電気敏感製品 | 導電性・帯電防止材を使用。電子部品に必須 |

## 3. 記載項目

### ヘッダー情報

| 項目 | 内容 |
|------|------|
| 梱包仕様書番号 / 版数 | 文書の識別番号と改訂版数 |
| 対象製品 | 製品名・品番・製品外形・重量 |
| 仕向地 | 国内 / 輸出先（国・地域） |
| 作成日 / 改訂日 | 初版日と最新改訂日 |
| 作成者 / 承認者 | 担当者と承認責任者 |
| 顧客承認 | 顧客の梱包承認状況（PPAP対象の場合） |

### 本体（梱包仕様）

#### A. 製品情報

| 項目 | 説明 |
|------|------|
| 製品名 / 品番 | 対象製品の識別 |
| 製品外形寸法 | 幅×奥行×高さ（mm） |
| 製品重量 | 単体重量（kg） |
| 脆弱部位 | 輸送中に損傷しやすい箇所の特定 |
| 環境感度 | 静電気・湿度・温度への敏感性 |

#### B. 一次梱包（個装）

| 項目 | 説明 |
|------|------|
| 包装材料 | 製品を直接包むフィルム・袋等の材質・寸法 |
| 緩衝材 | 製品を保護する緩衝材の材質・形状・配置 |
| 固定方法 | 製品を梱包内で固定する方法 |
| ESD対策 | 静電気対策が必要な場合の帯電防止材の仕様 |
| 防湿対策 | 防湿が必要な場合の包装材・乾燥剤の仕様 |

#### C. 二次梱包（内装）

| 項目 | 説明 |
|------|------|
| 内装箱の材質・寸法 | 段ボール等の材質・フルート・寸法 |
| 内装数量 | 1箱あたりの製品収納数 |
| 仕切り・トレー | 製品同士の接触を防ぐ仕切りの仕様 |
| 封緘方法 | テープ・ステープル等の封緘方法 |
| 内装箱の重量 | 製品を含む内装箱1箱の総重量 |

#### D. 三次梱包（外装）

| 項目 | 説明 |
|------|------|
| パレットサイズ | パレットの寸法（例: 1,100×1,100mm JIS T11型） |
| 段積みパターン | パレット上の内装箱の配列パターン（段数・列数） |
| 段積み数制限 | パレット上の最大段積み高さ・段数 |
| 固定方法 | ストレッチフィルム・PPバンド等の固定方法 |
| パレット当たり数量 | 1パレットあたりの製品総数 |
| 総重量 | パレット込みの総重量 |

#### E. 表示・ラベル

| 項目 | 説明 |
|------|------|
| 製品ラベル | 品番・製造日・ロット番号・シリアル番号・バーコード |
| 内装箱ラベル | 品番・数量・重量・製造日・ロット番号 |
| 外装ラベル | 品番・数量・重量・仕向地・出荷日 |
| 注意表示 | 取扱い注意マーク（天地無用、水濡れ注意、精密機器等） |
| 法規制表示 | CE、FCC、PSE等の法規制マーク |

#### F. 輸送・保管条件

| 項目 | 説明 |
|------|------|
| 輸送手段 | トラック / 鉄道 / 航空 / 海上 |
| 輸送中の環境条件 | 温度・湿度の許容範囲 |
| 保管環境 | 倉庫の温度・湿度条件、直射日光の回避等 |
| 保管期間 | 梱包状態での最大保管期間 |
| 段積み制限 | 倉庫内でのパレット段積み上限 |

## 4. 設計手順

### Step 1: 要求事項の整理

| インプット | 抽出する情報 |
|-----------|------------|
| 製品仕様書 | 製品の外形・重量・脆弱部位・環境感度 |
| 顧客要求 | 顧客固有の梱包基準・ラベル要件・納入形態 |
| 輸送条件 | 輸送手段・輸送距離・経由地の環境条件 |
| PFMEA | 輸送・保管中のリスク（振動、落下、温湿度変化等） |
| 法規制 | 輸出入規制、環境規制（梱包材のリサイクル義務等） |
| 物流計画 | トラック・コンテナのサイズ、パレットサイズ |
| コスト目標 | 梱包材コスト・物流コストの目標値 |

### Step 2: 梱包設計の要件定義

**輸送環境の想定**:

| 輸送ストレス | 条件 | 設計基準例 |
|------------|------|----------|
| 落下 | トラック荷台からの落下 | 高さ 76cm からの自由落下（ISTA 2A） |
| 振動 | トラック輸送中の振動 | ランダム振動 0.5G、30分間（ISTA 2A） |
| 圧縮 | 段積み時の荷重 | 最上段の3倍の荷重に耐えること |
| 温湿度 | 輸送・保管中の環境 | -10℃～+50℃、20～90%RH |
| 衝撃 | 荷扱い中の衝撃 | 水平衝撃 1.5G |

### Step 3: 梱包構造の設計

```
製品の脆弱性分析
    │
    ▼
緩衝設計（製品の許容Gから緩衝材の材質・厚みを算出）
    │
    ▼
一次梱包の設計（個装材料・緩衝材の配置）
    │
    ▼
二次梱包の設計（内装箱の設計・仕切り）
    │
    ▼
三次梱包の設計（パレット積載パターン・固定方法）
    │
    ▼
輸送試験による検証
```

**緩衝設計の基本**:

```
製品の許容G値（損傷限界加速度）
    │
    ▼
落下高さに対する必要な緩衝係数を計算
    │
    ▼
緩衝材の材質・密度・厚みを選定
    │
    ▼
                   許容G値の例
    ─────────────────────────
    精密機器:    15～25 G
    電子機器:    40～60 G
    一般製品:    60～80 G
```

### Step 4: 輸送試験

梱包設計の妥当性を輸送試験で検証します。

| 試験項目 | 試験規格 | 内容 |
|---------|---------|------|
| 落下試験 | ISTA 2A / JIS Z 0202 | 規定高さからの自由落下（面・辺・角） |
| 振動試験 | ISTA 2A / JIS Z 0232 | ランダム振動または正弦振動 |
| 圧縮試験 | JIS Z 0212 | 段積み荷重に相当する圧縮力 |
| 温湿度試験 | JIS Z 0208 | 高温・高湿環境での保管試験 |
| 水平衝撃試験 | JIS Z 0205 | 輸送中の急ブレーキを想定した衝撃 |

**合格基準**:
- 試験後に製品を開梱し、外観・機能に異常がないこと
- 製品の仕様（機能・性能）をすべて満たしていること

### Step 5: 梱包材の調達・標準化

- 梱包材の材質・寸法を標準化し、共通部材の活用を推進する
- 梱包材サプライヤーの品質管理を含めた調達管理を行う
- 環境規制への適合を確認する（RoHS、REACH、各国の梱包材規制）

### Step 6: レビューと承認

| 参加者 | レビュー観点 |
|--------|------------|
| 生産技術 | 梱包工程の作業性、梱包設備との適合性 |
| 品質保証 | 製品保護の十分性、輸送試験結果の妥当性 |
| 物流 | 積載効率、輸送コスト、保管効率 |
| 購買 | 梱包材コスト、調達リードタイム |
| 顧客（PPAP対象時） | 顧客固有の梱包基準への適合確認 |

### 梱包工程と他の工程設計文書との対応

| 文書 | 梱包に関連する記載内容 |
|------|---------------------|
| 工程フローチャート | 工程No.130 梱包を工程として記載 |

## 6. よくある課題と対策

### 課題1: 輸送中の製品損傷

**問題**: 輸送中に製品が破損し、顧客クレームが発生する。

**対策**:
- 輸送試験（ISTA 2A等）を実施し、梱包設計の妥当性を客観的に検証する
- 製品の許容G値から緩衝材の材質・厚みを計算し、設計根拠を明確にする
- 輸送ルートの環境条件（温度・振動レベル）を調査し、梱包設計に反映する
- 損傷発生時は原因調査を行い、梱包設計にフィードバックする

### 課題2: 梱包コストが高い

**問題**: 過剰な梱包材の使用により、梱包コストが製品コストに対して高い。

**対策**:
- 緩衝設計を定量的に行い、必要最小限の緩衝材量を算出する
- 梱包材の共通化・標準化を推進し、スケールメリットを活かす
- 通い箱（リターナブル）の導入を検討し、使い捨て梱包材を削減する
- 梱包構造の簡素化（ピース数の削減）を検討する

### 課題3: パレット積載効率が悪い

**問題**: 内装箱のサイズがパレットに対して最適化されておらず、積載効率が低い。

**対策**:
- パレットサイズ（1,100×1,100mm等）から逆算して内装箱のサイズを設計する
- 内装箱の配列パターンを複数案検討し、最も積載効率の高いパターンを選定する
- トラック・コンテナの内寸も考慮し、パレット配置も最適化する

### 課題4: 環境規制への対応

**問題**: 輸出先の梱包材規制（EU包装廃棄物指令等）に適合していない。

**対策**:
- 仕向地ごとの環境規制を事前に調査し、梱包仕様書に反映する
- リサイクル可能な材料（段ボール、EPS等）を優先的に使用する
- 木製パレット・木箱の場合はISPM15（燻蒸処理）への適合を確認する
- 梱包材の重金属含有規制（94/62/EC等）への適合を確認する

### 課題5: 梱包仕様書と実際の梱包作業が乖離

**問題**: 梱包仕様書で規定した方法と異なる梱包が行われている。

**対策**:
- 梱包仕様書を基に梱包作業指示書を作成し、作業者にわかりやすい手順として展開する
- 梱包工程をコントロールプランの管理対象に含め、定期的に確認する
- 梱包材のセットをキット化し、必要な材料が漏れなく供給されるようにする

## 7. ベストプラクティス

1. **定量的な緩衝設計を行う**: 製品の許容G値と輸送環境の衝撃レベルから、緩衝材の材質・厚みを計算で決定する
2. **輸送試験で設計を検証する**: ISTA・JIS等の標準試験規格に基づき、梱包設計の妥当性を客観的に検証する
3. **パレット効率から逆算して箱サイズを決める**: 物流効率を最大化するために、パレット→トラック→コンテナの順にサイズを最適化する
4. **環境規制を設計初期に確認する**: 仕向地の梱包材規制を事前に把握し、設計制約条件として取り込む
5. **梱包材を標準化する**: 品種間で梱包材を共通化し、在庫管理の簡素化とコスト削減を実現する
6. **通い箱の導入を検討する**: 繰返し使用可能なリターナブル梱包により、環境負荷とランニングコストを削減する
7. **梱包工程をPFMEA・CPに含める**: 梱包工程のリスク分析と管理計画を他の工程と同様に策定する
8. **顧客要求を早期に確認する**: PPAP対象の場合、顧客固有の梱包基準を設計初期に入手して設計に反映する
