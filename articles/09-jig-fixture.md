# 治工具設計図（Jig & Fixture Design Drawing）

## 1. 概要

### 治工具設計図とは

治工具設計図（Jig & Fixture Design Drawing）とは、製造工程で使用する**治具（Jig）・取付具（Fixture）・工具（Tool）・ゲージ（Gauge）** の形状・寸法・材質・精度を定義した設計図面です。

治工具は、製品の位置決め・固定・加工補助・検査補助を行う専用装置であり、**作業の再現性・品質の安定性・生産効率** を確保するうえで欠かせない存在です。治工具設計図はこれらの治工具を製作するための設計根拠文書であり、設備仕様書で定義された設備に合わせて設計されます。

### 目的

| 目的 | 説明 |
|------|------|
| 品質の安定化 | ワークの位置決め・固定を機械的に行い、加工・組立精度のばらつきを低減する |
| 作業の標準化 | 作業者の技量に依存しない均一な作業手順を実現する |
| ポカヨケの実現 | 誤品・誤方向・誤セットを物理的に防止する構造を設計に織り込む |
| 生産性の向上 | ワークの着脱時間を短縮し、サイクルタイム内で安定した生産を可能にする |
| 製作・保全の根拠 | 治工具の製作指示・修理・再製作時に必要な設計情報を文書として残す |
| 変更管理 | 治工具の改造・変更時に元の設計との差分を明確にし、品質影響を評価する |

### 工程設計における位置づけ

```
工程フローチャート（各工程の作業内容を定義）
    │
    ▼
設備仕様書（各工程で必要な設備の要求仕様を定義）
    │
    ▼
治工具設計図（設備に合わせた治具・工具・ゲージを設計） ◀ 本記事
    │
    ├──→ 治工具の製作・調達
    │
    ├──→ フロアプランレイアウト（設備・治工具を配置する平面計画）
    │
    └──→ 作業指示書（治工具の使用方法を手順に記載）
            │
            ▼
        コントロールプラン / QC工程表（治工具の管理方法を規定）
```

治工具設計図は設備仕様書の下流に位置し、設備に適合した治具・工具を具体的な図面として設計する文書です。PFMEAで特定されたリスク低減策（ポカヨケ等）を物理的な構造として実現する役割を担います。

> **補足**: 組織によっては、治工具設計図の上流文書として**工具要求仕様書**（Tool Requirement Specification）を発行する場合があります。工程設計技術課が「何を達成すべきか（機能要求・性能要求・受入条件）」を定義し、生産技術課が設計を担う分業体制において特に有効です。詳細は [工具要求仕様書](./09a-tool-requirement-spec.md) を参照してください。

### IATF 16949の要求事項

IATF 16949:2016では、治工具に関連する主な要求条項は以下のとおりです。

| 条項 | 内容 |
|------|------|
| **8.5.1.5** 生産ツールのマネジメント | 工具・ゲージ・治具の設計・製作・検証の仕組みを確立すること |
| **8.5.1.6** 生産ツーリングの管理 | 工具・ゲージの識別・保管・修理・廃棄の管理を行うこと |
| **8.3.3.3** 特殊特性 | 特殊特性（CC/SC）の管理に必要なポカヨケ等を設計に含めること |
| **8.5.1.3** 作業セットアップの検証 | 治工具のセットアップ時に初品検証を行うこと |

## 2. 治工具の分類

### 基本分類

工程設計で扱う治工具は、機能・用途に応じて以下のように分類されます。

| 分類 | 英語名 | 機能 | 例 |
|------|--------|------|-----|
| **治具** | Jig | ワークの位置決め＋加工ガイド（ドリルブッシュ等）を提供する装置 | ドリルジグ、溶接ジグ、ルーティングジグ |
| **取付具** | Fixture | ワークを位置決め・固定（クランプ）する装置（加工ガイドは持たない） | 組立固定治具、溶接固定治具、検査固定治具 |
| **専用工具** | Special Tool | 特定の工程専用に設計された作業補助ツール | 圧入工具、カシメ工具、挿入ガイド |
| **検査ゲージ** | Gauge | 製品の合否判定を行うための専用検査具 | 限界ゲージ（Go/No-Go）、検査治具、マスターサンプル |
| **段取り治具** | Setup Tool | 設備の段取り替え時に使用する補助具 | 位置決めマスター、ゼロ点校正ゲージ |

> **注記**: 日本語の「治工具」は「治具」と「工具」の総称です。英語では Jig と Fixture は明確に区別されますが、日本の現場では「治具」と総称することが多いです。本記事では治具・取付具・専用工具・検査ゲージを含む広義の「治工具」を対象とします。

### 工程種別ごとの代表的な治工具

| 工程種別 | 治工具の種類 | 主な機能 |
|----------|------------|---------|
| 組立工程 | 組立治具 | 部品同士の位置関係を拘束して組み付ける |
| 加工工程 | 加工治具 | ワークを固定し加工基準面を提供する |
| 溶接工程 | 溶接治具 | 溶接部品の相対位置を保持する |
| 検査工程 | 検査治具・ゲージ | 製品の寸法・形状の合否を判定する |
| 搬送工程 | 搬送パレット・トレー | 製品を定位置に保持して搬送する |
| 圧入工程 | 圧入治具 | 圧入方向・圧入量をガイドする |
| 挿入工程 | 挿入ガイド治具 | 部品の挿入方向と嵌合をガイドする |

## 3. 治工具設計の基本原則

### 3-2-1原則（位置決めの基本）

治工具における位置決めの基本原則は**3-2-1原則**です。直方体のワークを空間上で完全に拘束するためには、6つの自由度（X・Y・Z方向の並進3軸＋回転3軸）を拘束する必要があります。

```
拘束される自由度     位置決め方法          ロケーターの数
───────────    ──────────────    ──────────
Z方向 並進 + X回転 + Y回転   底面（主基準面）に3点ロケーター    3個
Y方向 並進 + Z回転           側面（第2基準面）に2点ロケーター    2個
X方向 並進                   端面（第3基準面）に1点ロケーター    1個
──────────────────────────────────────────    合計 6個
```

```
        Z
        ↑
        │    ┌───────────────┐
        │   /│              /│
        │  / │     ③端面   / │
        │ /  │    （1点）  /  │
        │/   │           /   │
        ├───────────────┤    │
        │    │          │    │
        │    │  ②側面   │    │
        │    │ （2点）   │    │
        │    └─────────│────┘──→ Y
        │   /           │   /
        │  /   ①底面    │  /
        │ /   （3点）    │ /
        │/               │/
        └────────────────┘
       /
      X
```

### クランプの原則

| 原則 | 内容 |
|------|------|
| ロケーターに対向させる | クランプ力はロケーター（位置決め点）の方向に向ける |
| ワークを変形させない | クランプ力が過大にならないよう設計する |
| 着脱容易性 | 片手操作・ワンタッチ着脱を目指す（サイクルタイム短縮） |
| 再現性 | 繰返しセットで同じ位置に固定できる構造とする |

### ポカヨケ設計

PFMEAで特定した故障モードに対し、治工具レベルで防止するポカヨケ構造を組み込みます。

| ポカヨケの種類 | 方法 | 例 |
|--------------|------|-----|
| 誤方向防止 | 非対称形状のロケーター配置 | ワークの向きを1方向のみセット可能にする |
| 誤品防止 | 品番固有の形状・寸法の利用 | 他品番がセットできない形状にする |
| 欠品検知 | 部品有無のセンサー検知 | 近接センサーで部品の有無を確認する |
| 未加工品検知 | 前工程の加工痕を検知 | ピンが前工程の穴に嵌合するか確認する |
| 二重セット防止 | 高さ検知 | ワークが二重にセットされると高さセンサーでNG |

| 項目 | 内容 |

| 部品番号（バルーン） | 各部品に付した識別番号 |
| 主要寸法 | 治工具の外形寸法・設備取付寸法 |

#### B. 部品図（詳細図）

| 項目 | 説明 |
|------|------|
| 個別部品図 | 各部品の形状・寸法を定義した図面 |

| 表面粗さ | 加工面の表面粗さ指定（Ra値等） |
| 材質 | 部品の材質指定（SS400、S45C、SKD11等） |
| 熱処理 | 焼入れ・焼戻し・窒化処理等の指定 |
| 表面処理 | メッキ・塗装・黒染等の指定 |

#### C. 部品表（Parts List）

| 項目 | 説明 |
|------|------|
| 部品番号 | 部品の識別番号 |
| 部品名称 | 部品の名称 |
| 材質 | 使用する材料 |
| 数量 | 必要個数 |
| 購入品 / 製作品 | 購入品（ボルト・ピン等）か製作品かの区分 |
| 備考 | 市販品の場合はメーカー・型番を記載 |

#### D. 精度要件

| 項目 | 説明 |
|------|------|
| 位置決め精度 | ワークのロケーター位置の精度（位置度、平行度等） |
| 繰返し精度 | ワークの着脱を繰り返した際の位置決め精度のばらつき |
| 設備取付精度 | 設備への取付時の位置合わせ精度 |
| 検査ゲージ精度 | 検査治具の場合、判定精度（Go/No-Go寸法の公差） |

#### E. 注記事項

| 項目 | 説明 |
|------|------|
| ポカヨケ機能 | 誤方向防止・誤品防止等の設計意図を注記 |
| 使用上の注意 | 取扱い上の注意事項（最大荷重、使用温度等） |
| 保全・校正 | 摩耗部品の交換基準、校正頻度 |
| 関連文書 | 対応する設備仕様書・コントロールプラン・作業指示書の番号 |

## 5. 設計手順

### Step 1: 要求事項の整理

工程フローチャート・PFMEA・設備仕様書をインプットに、治工具に対する要求事項を整理します。

| インプット | 抽出する情報 |
|-----------|------------|
| 工具要求仕様書 | 機能要求・性能要求・受入条件（工程設計技術課発行の場合） |
| 工程フローチャート | 各工程の作業内容、使用する治工具の概要 |
| PFMEA | 故障モードと対策（ポカヨケ要件、位置決め精度要件） |
| コントロールプラン | 管理特性、精度要件、特殊特性（CC/SC） |
| 設備仕様書 | 設備のワーク搬入方式、設備側の取付インターフェース |
| 製品図面 | ワークの形状・寸法・材質・重量、基準面の指定 |
| 生産計画 | サイクルタイム要件（着脱時間の目標） |

> 工程設計技術課と生産技術課が分業する体制では、[工具要求仕様書](./09a-tool-requirement-spec.md) が本ステップの主要インプットとなります。要求仕様書に記載された機能要求・性能要求・受入条件を設計の出発点として活用してください。

### Step 2: 構想設計

要求事項をもとに、治工具の構想を設計します。

**検討項目**:

| 項目 | 内容 |
|------|------|
| 位置決め方式 | 3-2-1原則に基づくロケーター配置の決定 |
| クランプ方式 | 手動クランプ / 空圧クランプ / トグルクランプ等の選定 |
| ポカヨケ構造 | PFMEAの故障モードに対する防止構造の設計 |
| 着脱方式 | ワンタッチ着脱 / スライド着脱等の方式 |
| 搬送適合性 | コンベア・ロボット等の搬送方式との適合確認 |
| 設備インターフェース | 設備への取付方法・位置合わせ方法 |

**構想設計時のポイント**:

```
製品図面の基準面     治工具の位置決め基準
──────────────   ──────────────
製品の設計基準（データム）を確認
    │
    ▼
工程基準（加工・組立基準）を決定
    │
    ▼
3-2-1原則に基づくロケーター配置を決定
    │
    ▼
クランプ位置・方式を決定
    │
    ▼
ポカヨケ構造を組み込み
    │
    ▼
構想図（スケッチ・3D CAD）を作成
```

### Step 3: 詳細設計

構想設計をもとに、製作可能な詳細図面を作成します。

- 各部品の形状・寸法・公差を決定する
- 材質・熱処理・表面処理を決定する
- 購入品（ボルト・ピン・クランプ金具等）を選定する
- 組立図・部品図・部品表を作成する

**精度設計の原則**:

```
製品の品質特性の公差        治工具の精度要件
──────────────     ─────────────
公差の 1/5 ～ 1/10    →   治工具の位置決め精度（一般的な目安）
公差の 1/3 以下        →   検査ゲージの判定精度（JIS B 7420 等）
```

**材質選定の指針**:

| 部品の種類 | 推奨材質 | 硬度目安 | 理由 |
|-----------|---------|---------|------|
| ロケーターピン | SKD11, SUJ2 | HRC 58-62 | 繰返し接触による摩耗に耐える |
| クランプ部品 | S45C（焼入れ） | HRC 40-45 | 適度な強度と靭性 |
| ベースプレート | SS400, S50C | 生材 or HRC 25-30 | 剛性を確保しつつコスト低減 |
| ワーク接触面 | 樹脂（MCナイロン等） | - | ワーク表面のキズ防止 |
| 検査ゲージ面 | SKD11, 超硬合金 | HRC 60以上 | 長期間の寸法安定性 |

### Step 4: 設計レビュー

| 参加者 | レビュー観点 |
|--------|------------|
| 生産技術（設計者） | 設計意図、精度計算の妥当性 |
| 製造 | 操作性、着脱のしやすさ、サイクルタイムへの影響 |
| 品質保証 | ポカヨケ機能の十分性、特殊特性への対応 |
| 保全 | 摩耗部品の交換容易性、保全のしやすさ |
| 治工具製作先 | 製作可能性、コスト、納期 |

### Step 5: 製作・検収

治工具を製作し、精度・機能を検証します。

```
詳細設計図面の完成
    │
    ▼
治工具の製作（社内 or 外注）
    │
    ▼
精度検査（寸法測定・幾何公差の検証）
    │
    ▼
機能確認（ワークセット・着脱テスト・ポカヨケ動作確認）
    │
    ▼
設備への取付・調整
    │
    ▼
量産トライアル（連続稼働での安定性確認）
    │
    ▼
検収完了 → 治工具台帳に登録
```

### Step 6: 治工具台帳への登録

完成した治工具を管理台帳に登録し、ライフサイクルを通じた管理を開始します。

| 登録項目 | 内容 |
|---------|------|
| 治工具管理番号 | 社内識別番号 |
| 治工具名称 | 正式名称 |
| 設計図番号 | 対応する設計図面番号 |
| 対象工程・設備 | 使用工程と対応設備 |
| 製作日 | 初回製作完了日 |
| 製作先 | 社内 / 外注先名 |
| 使用回数 / 寿命管理 | ショット数管理等の寿命基準 |
| 校正周期 | 検査ゲージの場合の校正頻度 |
| 保管場所 | 治工具保管棚の識別 |

## 6. 具体例

### インクジェットプリンタ PX-500 組立工程の治工具設計図

工程No.70（基板実装）で使用する**コネクタ挿入ガイド治具**の設計図を例示します。

```
─────────────────────────────────────────────────────────────
治工具設計図

図面番号: JG-PX500-070-01-R01        改訂日: 2025/12/15
治工具名称: コネクタ挿入ガイド治具
治工具管理番号: JG-070
対象工程: 工程No.70 基板実装
対応設備: 電動ドライバー EQ-070
関連CP: CP-PX500-R01（特性No.7 コネクタ嵌合力）
関連PFMEA: FM-PX500-R01（工程No.70）

設計: 生産技術 鈴木    承認: 生産技術部長 佐藤
製作先: 社内 治工具室
─────────────────────────────────────────────────────────────

■ 目的・用途
  工程No.70基板実装において、フレキシブルフラットケーブル（FFC）
  コネクタを制御基板に挿入する際に、挿入方向・挿入量をガイドし、
  規定の嵌合力（15±3N）で均一に挿入されることを補助する。

■ 対象製品
  製品名: インクジェットプリンタ PX-500
  品番: PX-500A
  対象部品: FFCコネクタ（品番: CN-070A）
  対象基板: メイン制御基板（品番: PCB-070A）

■ 管理特性
  ◇SC: コネクタ嵌合力 15±3N

─────────────────────────────────────────────────────────────
■ A. 設計要件
─────────────────────────────────────────────────────────────

  位置決め方式（3-2-1原則に基づく）
```

## 7. 治工具のライフサイクル管理

治工具は製作後も継続的な管理が必要です。IATF 16949の「8.5.1.6 生産ツーリングの管理」に基づき、ライフサイクル全体を管理します。

### ライフサイクルフロー

```
設計 → 製作 → 検収 → 使用開始 → 定期点検・校正
                                    │
                    ┌───────────────┤
                    │               │
                    ▼               ▼
                  修理・再製作    摩耗・寿命到達
                    │               │
                    ▼               ▼
                  再検収          廃棄 → 新規治工具設計
```

### 管理項目

| 管理項目 | 内容 | 頻度 |
|---------|------|------|
| 日常点検 | 外観確認（キズ・変形・摩耗）、動作確認 | 毎日（使用前） |
| 定期精度確認 | 位置決め精度・ゲージ寸法の測定 | 月1回（または規定ショット数ごと） |
| 校正 | 検査ゲージの校正（マスターとの比較） | 月1回～年1回（治工具種類による） |
| 摩耗部品交換 | ロケーターピン・ガイドブロック等の交換 | 交換基準値到達時 |
| 識別管理 | 管理番号ラベル・校正期限ラベルの貼付 | 常時 |
| 保管管理 | 専用棚への保管、防錆処理 | 使用後毎回 |
| 廃棄管理 | 寿命到達・不要時の廃棄記録 | 廃棄時 |

## 8. よくある課題と対策

### 課題1: 治工具精度が製品精度に対して不足

**問題**: 治工具の位置決め精度が不十分なため、製品のばらつきが大きくなる。

**対策**:
- 製品の品質特性の公差から逆算して治工具の精度を設定する（公差の1/5～1/10が目安）
- PFMEA・コントロールプランの精度要件と治工具設計図の精度要件を照合する
- 治工具検収時に精度測定を必ず実施し、記録を残す

### 課題2: ワークの着脱に時間がかかる

**問題**: 治工具のクランプが複雑で着脱時間が長く、サイクルタイムを圧迫する。

**対策**:
- トグルクランプ・空圧クランプ等のワンアクション機構を採用する
- 「片手操作」を設計基準として設定する
- 構想設計段階で製造担当者の意見を反映する

### 課題3: ポカヨケが不十分

**問題**: 治工具にポカヨケ構造がなく、ワークの誤方向セットや誤品セットが発生する。

**対策**:
- PFMEAで特定した故障モード（誤セット・欠品等）に対応するポカヨケ構造を治工具設計に必ず組み込む
- 非対称形状の利用、センサー検知、高さ検知等の手法を設計チェックリストに含める
- 特殊特性（CC/SC）に関連する工程では、ポカヨケ設計を必須とする

### 課題4: 治工具の摩耗管理が不十分

**問題**: ロケーターピンやガイドブロックの摩耗が放置され、位置決め精度が劣化する。

**対策**:
- 摩耗部品ごとに交換基準（摩耗量の上限値）を治工具設計図に明記する
- 定期点検のチェックシートを作成し、精度測定結果を記録する
- 摩耗傾向を分析し、予防的な部品交換計画を策定する

### 課題5: 治工具設計図が改訂されない

**問題**: 治工具を改造・修理した後に設計図が更新されず、図面と実物が乖離する。

**対策**:
- 治工具の変更管理プロセスに設計図の改訂を組み込む
- 変更時にはコントロールプラン・作業指示書との整合確認を行う
- 治工具台帳と設計図の版数を定期的に照合する

## 9. ベストプラクティス

1. **3-2-1原則を忠実に適用する**: 位置決めの基本原則に基づき、ロケーターの数と配置を設計する。過剰拘束（過定位）は精度不良の原因となるため、必要最小限のロケーターとする
2. **PFMEAと連動してポカヨケを設計する**: PFMEAで特定した故障モードに対して、治工具レベルで防止する構造を必ず組み込む
3. **着脱時間をサイクルタイムの中で計画する**: 治工具の着脱時間を含めてサイクルタイムが達成できるよう、構想設計段階から時間を考慮する
4. **摩耗部品の交換基準を設計図に明記する**: ロケーターピン・ガイドブロック等の摩耗部品には、交換基準値（摩耗量の上限）を図面に記載する
5. **製造担当者を設計レビューに参加させる**: 実際に治工具を使用する作業者の意見を設計に反映し、操作性を確保する
6. **ワーク接触面にキズ防止策を講じる**: 外観品質に影響する面に治工具が接触する場合、樹脂パッド・フェルト等の緩衝材を使用する
7. **3D CADとシミュレーションを活用する**: 干渉チェック・応力解析・動作シミュレーションにより、製作前に問題を検出する
8. **標準部品を積極的に活用する**: ロケーターピン・クランプ金具等は市販の標準部品を採用し、コスト低減と部品調達の容易性を確保する
