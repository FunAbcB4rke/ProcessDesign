# フロアプランレイアウト（Floor Plan Layout）

## 1. 概要

### フロアプランレイアウトとは

フロアプランレイアウト（Floor Plan Layout）とは、生産ラインにおける**設備・治工具・作業ステーション・搬送経路・保管エリア**の配置を平面図として設計した文書です。

製品が受入から出荷まで効率的かつ安全に流れるよう、物の流れ（マテリアルフロー）と人の動き（ワーカーフロー）を最適化することを目的とします。工程フローチャートで定義した工程の流れを物理的な空間に落とし込む文書であり、生産効率・品質・安全性のすべてに直結します。

### 目的

| 目的 | 説明 |
|------|------|
| 物流の最適化 | 製品・部品・材料の移動距離を最小化し、搬送ロスを削減する |
| 工程間の整流化 | 工程の流れに沿った設備配置で、逆流・交差・滞留を排除する |
| 作業効率の向上 | 作業者の動線を短縮し、付加価値のない歩行・運搬を削減する |
| 品質リスクの低減 | 不良品の混入・取り違えが起きにくい物理的な配置を実現する |
| 安全の確保 | 人と設備・搬送機器の動線を分離し、衝突・挟まれ等のリスクを低減する |
| スペースの有効活用 | 生産面積あたりの生産性を最大化し、投資効率を高める |
| 将来の拡張性 | 増産・品種追加・設備更新に対応できる余裕スペースを確保する |

### 工程設計における位置づけ

```
工程フローチャート（各工程の作業内容・順序を定義）
    │
    ├──→ 設備仕様書（各工程で必要な設備を定義）
    │        │
    │        ├──→ 治工具設計図（設備に合わせた治工具を設計）
    │        │
    │        └──→ フロアプランレイアウト（設備・治工具を配置する平面計画） ◀ 本記事
    │                  │
    │                  ├──→ 搬送設備の設計（コンベア・AGV等）
    │                  │
    │                  ├──→ ユーティリティ配管・配線設計
    │                  │
    │                  └──→ 安全設備の配置（安全柵・非常停止等）
    │
    └──→ コントロールプラン / 作業指示書
```

フロアプランレイアウトは、工程フローチャートの論理的な流れと設備仕様書の物理的な要件を空間設計に統合する文書です。設備の外形寸法・ユーティリティ要件・保全スペースを考慮し、最適な配置を決定します。

### IATF 16949の要求事項

| 条項 | 内容 |
|------|------|
| **7.1.3** インフラストラクチャ | 製品の適合を達成するために必要な施設（建屋・作業スペース・関連ユーティリティ）を決定し、提供・維持すること |
| **7.1.4** プロセスの運用環境 | 製品の適合を達成するために必要な環境（温度・湿度・清浄度・照明等）を決定し、維持すること |
| **8.3.3.1** 製品設計へのインプット | 製造容易性（Manufacturability）をインプットに含めること |
| **8.5.4** 保存 | 製品の特性を維持する適切な保管条件を提供すること |

## 2. レイアウト形式の分類

### 基本レイアウト形式

生産ラインのレイアウトは、生産形態・生産量・品種数に応じて以下の形式に分類されます。

| 形式 | 英語名 | 特徴 | 適用場面 |
|------|--------|------|---------|
| **ライン型（I型）** | Line Layout | 直線配置。製品が一方向に流れる | 大量生産、品種が少ない |
| **U字型** | U-shaped Layout | 入口と出口を同じ側に配置。作業者が複数工程を担当しやすい | セル生産、多能工化 |
| **L字型** | L-shaped Layout | スペース制約に対応した直角配置 | 工場スペースの制約がある場合 |
| **S字型（蛇行型）** | S-shaped Layout | 限られたスペースで長い工程を折り返して配置 | 工程数が多くスペースが限られる場合 |
| **セル型** | Cell Layout | 関連工程をグループ化して配置 | 多品種少量、部品ファミリー生産 |
| **機能別配置** | Process Layout | 同種の設備をまとめて配置 | ジョブショップ、多品種変量 |

### 各形式の比較

```
■ ライン型（I型）
  IN → [工程1] → [工程2] → [工程3] → [工程4] → [工程5] → OUT

■ U字型
  IN → [工程1] → [工程2] → [工程3]
                                ↓
  OUT ← [工程6] ← [工程5] ← [工程4]

■ L字型
  IN → [工程1] → [工程2] → [工程3]
                                ↓
                            [工程4]
                                ↓
                            [工程5] → OUT

■ S字型（蛇行型）
  IN → [工程1] → [工程2] → [工程3]
                                ↓
       [工程6] ← [工程5] ← [工程4]
         ↓
       [工程7] → [工程8] → OUT
```

### レイアウト形式の選定基準

| 評価項目 | ライン型 | U字型 | セル型 |
|---------|---------|-------|--------|
| 大量生産への適合 | ◎ | ○ | △ |
| 多品種対応 | △ | ○ | ◎ |
| 作業者の柔軟配置 | △ | ◎ | ◎ |
| スペース効率 | △ | ◎ | ○ |
| 工程間仕掛り | 多い | 少ない | 少ない |
| 生産量変動への対応 | △ | ◎ | ◎ |

## 3. 設計の基本原則

### 整流化の原則

製品の流れを**一方向**に整流化し、逆流・交差・滞留を排除します。

| 原則 | 内容 |
|------|------|
| 一方向の流れ | 受入 → 加工 → 組立 → 検査 → 梱包 → 出荷の順に一方向に配置する |
| 逆流の排除 | 前工程に戻る搬送が発生しないレイアウトにする |
| 交差の排除 | 物流動線同士が交差しない配置にする |
| 滞留の最小化 | 工程間バッファ（仕掛り）を最小限にする |
| 搬送距離の最小化 | 隣接する工程をできるだけ近くに配置する |

### 動線分離の原則

人の動線と物の動線を分離し、安全を確保します。

```
┌────────────────────────────────────────────┐
│          生産エリア                           │
│                                              │
│  [設備]─→[設備]─→[設備]  ← 物流動線（製品）  │
│                                              │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  ← 作業者通路       │
│                                              │
│  [部品棚]  [部品棚]  [部品棚] ← 部品供給動線  │
│                                              │
│  ═══════════════════════  ← フォークリフト通路 │
│          （人と分離）                          │
└────────────────────────────────────────────┘
```

### スペース計画の原則

| 項目 | 基準 |
|------|------|
| 設備間隔 | 設備外形 + 保全スペース（設備周囲 600mm以上） |
| 作業者通路 | 主通路 1,200mm以上、副通路 800mm以上 |
| フォークリフト通路 | 3,000～3,500mm（対向通行の場合） |
| AGV通路 | AGV幅 + 左右各 200mm以上 |
| 非常口への経路 | 避難距離 30m以内（消防法） |
| 部品供給スペース | ラインサイドに2時間分以上の部品在庫スペース |
| 不良品隔離エリア | 各工程の近くに赤札エリアを確保 |

## 4. 記載項目

### ヘッダー情報

| 項目 | 内容 |
|------|------|
| 図面番号 / 版数 | レイアウト図の識別番号と改訂版数 |
| 対象ライン名称 | 生産ラインの正式名称 |
| 対象製品 | 生産する製品の品番・品名 |
| 対象工場 / 棟 | 設置する工場・建屋の識別 |
| 作成日 / 改訂日 | 初版日と最新改訂日 |
| 作成者 / 承認者 | 担当者と承認責任者 |
| 縮尺 | 図面の縮尺（例: 1/50、1/100） |

### 本体（レイアウト図面）

#### A. 設備配置

| 項目 | 説明 |
|------|------|
| 設備の外形・位置 | 各設備の外形寸法と配置位置を縮尺図面で表示 |
| 設備管理番号 | 各設備の識別番号（設備仕様書と対応） |
| 工程番号 | 工程フローチャートの工程番号を設備に付記 |
| 設備の向き | 設備のワーク搬入面・操作面の方向を明示 |
| 固定設備 / 可動設備 | 移設可能な設備と固定設備を区別して表示 |

#### B. 動線計画

| 項目 | 説明 |
|------|------|
| 製品フロー | 主製品の流れ（矢印で方向を表示） |
| 部品供給ルート | 部品・材料の搬入ルート |
| 完成品搬出ルート | 完成品の搬出ルート |
| 作業者動線 | 作業者の歩行ルートと通路幅 |
| 搬送設備 | コンベア・AGV・台車のルートと停止位置 |
| 不良品・廃棄物動線 | 不良品の隔離場所への搬送ルート |

#### C. 付帯設備・エリア

| 項目 | 説明 |
|------|------|
| 部品棚・ラック | ラインサイドの部品保管棚の配置 |
| 仕掛りバッファ | 工程間バッファの位置と容量 |
| 不良品隔離エリア | 赤札品の一時保管場所 |
| 検査エリア | 中間検査・最終検査のスペース |
| ユーティリティ | 電源盤・エアー配管・排気ダクトの位置 |
| 安全設備 | 安全柵・非常停止ボタン・消火器の位置 |
| 表示・掲示物 | 工程表示板・作業指示書掲示位置・安全標識 |

#### D. 寸法情報

| 項目 | 説明 |
|------|------|
| 建屋寸法 | 生産エリアの全体寸法（幅×奥行×天井高） |
| 設備間距離 | 隣接設備間のクリアランス |
| 通路幅 | 各通路の幅員 |
| 柱・壁の位置 | 建屋の構造物の位置 |
| 床荷重 | 重量設備の設置箇所の耐荷重 |

## 5. 設計手順

### Step 1: インプット情報の収集

| インプット | 抽出する情報 |
|-----------|------------|
| 工程フローチャート | 全工程の順序、工程数、工程間の依存関係 |
| 設備仕様書 | 各設備の外形寸法・重量・ユーティリティ要件 |
| 治工具設計図 | 治工具の外形寸法、設備への取付位置 |
| 生産計画 | タクトタイム、生産量、シフト体制、将来の増産計画 |
| 製品情報 | 製品の外形・重量・搬送方法 |
| 工場情報 | 建屋の寸法・柱位置・床荷重・ユーティリティ供給位置 |
| 安全規制 | 消防法・労働安全衛生法の要件 |

### Step 2: 工程間の近接性分析

各工程間の近接性（どの工程同士を近くに配置すべきか）を分析します。

**近接性評価マトリクスの例**:

| 工程 | 受入 | 駆動部組立 | キャリッジ組立 | 基板実装 | 検査 | 梱包 |
|------|------|-----------|-------------|---------|------|------|
| 受入 | - | A | E | E | U | U |
| 駆動部組立 | A | - | A | I | U | U |
| キャリッジ組立 | E | A | - | A | I | U |
| 基板実装 | E | I | A | - | A | U |
| 検査 | U | U | I | A | - | A |
| 梱包 | U | U | U | U | A | - |

**評価基準**: A=絶対的に近接必要、E=特に重要、I=重要、O=普通、U=不要、X=離すべき

### Step 3: レイアウト形式の選定

生産条件に基づいてレイアウト形式を選定します。

| 検討項目 | 判断基準 |
|---------|---------|
| 生産量 | 大量→ライン型、少量→セル型 |
| 品種数 | 少品種→ライン型、多品種→セル/U字型 |
| 工程数 | 多い→S字型/U字型、少ない→I型 |
| スペース制約 | 長いスペース→I型、正方形に近い→U字型/セル型 |
| 多能工化方針 | 多能工→U字型/セル型 |
| 将来の拡張性 | 増設余地が必要→端部に拡張スペースを確保 |

### Step 4: ブロックレイアウトの作成

工程をブロック（区画）として配置し、全体の配置構想を決定します。

```
┌─────────────────────────────────────────────────┐
│                    生産エリア                      │
│                                                   │
│  ┌──────┐   ┌──────┐   ┌──────┐   ┌──────┐    │
│  │受入・  │   │組立    │   │組立    │   │検査    │    │
│  │保管    │ → │エリア  │ → │エリア  │ → │エリア  │    │
│  │エリア  │   │ (前半) │   │ (後半) │   │        │    │
│  └──────┘   └──────┘   └──────┘   └──────┘    │
│       ↑                                    ↓      │
│  ┌──────┐                          ┌──────┐    │
│  │部品倉庫 │                          │梱包・  │    │
│  │        │                          │出荷    │    │
│  └──────┘                          └──────┘    │
│                                                   │
│  ──────── 主通路 ──────────────────────          │
└─────────────────────────────────────────────────┘
```

### Step 5: 詳細レイアウトの作成

ブロックレイアウトを基に、各設備の正確な位置を配置します。

**検討項目**:

| 項目 | 内容 |
|------|------|
| 設備の向き | ワーク搬入面・操作面が作業者通路側を向くように配置 |
| 保全スペース | 設備の背面・側面に保全作業用のアクセス空間を確保 |
| ユーティリティ接続 | 電源・エアー・排気の供給口に近い位置に設備を配置 |
| 搬送手段 | コンベア・台車・AGVのルートと設備の接続方法 |
| 作業者の手元照明 | 検査工程・精密作業工程には十分な照度を確保 |
| 床面表示 | 設備位置・通路・安全区域の床面ペイント・テープ |

### Step 6: 動線の検証

配置した設備に対して、製品・部品・作業者の動線を検証します。

**検証項目**:

| 検証項目 | 確認内容 |
|---------|---------|
| 製品動線の距離 | 受入から出荷までの総搬送距離を計算し、最短化されているか確認 |
| 逆流・交差の有無 | 動線が交差・逆流していないか確認 |
| 搬送時間 | 工程間の搬送時間がタクトタイムに影響しないか確認 |
| 作業者動線 | 1サイクル中の作業者の歩行距離を計算 |
| 部品供給ルート | 部品搬入ルートが生産動線と干渉しないか確認 |
| 安全動線 | 非常口までの避難距離が規定内か確認 |
| フォークリフト動線 | 人の動線と交差しないか確認 |

### Step 7: レビューと承認

| 参加者 | レビュー観点 |
|--------|------------|
| 生産技術 | 設備配置の技術的妥当性、ユーティリティの適合性 |
| 製造 | 作業性、動線の効率性、作業者配置の柔軟性 |
| 品質保証 | 検査エリアの適切性、不良品隔離の動線 |
| 物流 | 部品供給・完成品搬出の効率性 |
| 安全衛生 | 安全基準への適合、避難経路の確保 |
| 施設管理 | 建屋制約（柱・床荷重・天井高）との適合、ユーティリティ容量 |
| 保全 | 設備のメンテナンスアクセス |

### U字型レイアウトを選定した理由

| 評価項目 | U字型の優位性（PX-500ラインの場合） |
|---------|-------------------------------|
| 入口/出口の近接 | 受入と出荷が同じ側にあり、部品供給と完成品搬出の効率が良い |
| 作業者の柔軟配置 | 生産量変動時に1人が対面の2工程を担当でき、少人化が可能 |
| コミュニケーション | U字の対面で作業者同士が見渡せ、異常発生時の連絡が早い |
| スペース効率 | 30m×10mの矩形に収まり、建屋のスペース制約に適合 |
| 仕掛り削減 | 工程間距離が短く、仕掛り在庫を最小化できる |

## 7. よくある課題と対策

### 課題1: 物流動線が交差する

**問題**: 製品の流れと部品供給ルート、または製品動線同士が交差し、滞留・混入リスクが発生する。

**対策**:
- 製品動線と部品供給動線を物理的に分離する（製品はライン内側、部品供給はライン外側から）
- フォークリフト通路と作業者通路を明確に分離し、床面表示で識別する
- 交差が不可避な場合は信号・ミラー・ゲート等で安全対策を講じる

### 課題2: 保全スペースが不足

**問題**: 設備の背面や側面に十分なスペースがなく、保全作業が困難になる。

**対策**:
- 設備仕様書の外形寸法に加え、保全アクセスに必要なスペース（600mm以上）を確保する
- 設備メーカーから保全時のアクセス方向・必要スペースの情報を入手してレイアウトに反映する
- 可動式の設備・治工具は移動可能スペースを考慮する

### 課題3: 将来の増産に対応できない

**問題**: レイアウトに余裕がなく、設備追加・ライン増設時に大幅な変更が必要になる。

**対策**:
- 初期計画の段階で将来の増産計画（例: 3年後に1.5倍）を考慮してスペースを確保する
- ラインの端部に拡張用スペースを設ける
- 設備配置をモジュール化し、増設しやすい構造にする

### 課題4: 作業者の歩行距離が長い

**問題**: 工程間の距離が長く、作業者が歩行に時間を費やしてタクトタイムを圧迫する。

**対策**:
- U字型・セル型レイアウトにより、作業者が複数工程を最短距離で担当できる配置にする
- 部品棚を手の届く範囲に配置し、部品を取りに歩く距離を削減する
- ワーク搬送にはコンベア・シューター等を活用し、作業者の搬送作業を削減する

### 課題5: 不良品の混入リスク

**問題**: 良品と不良品の動線が分離されておらず、不良品の混入リスクがある。

**対策**:
- 各工程に不良品隔離エリア（赤札エリア）を設ける
- 良品の搬送ルートと不良品の搬出ルートを物理的に分離する
- 不良品隔離エリアは製品動線から離れた位置に配置する

## 8. ベストプラクティス

1. **整流化を最優先にする**: 受入から出荷までの製品の流れを一方向にし、逆流・交差を排除する
2. **工程フローチャートとの一致を確認する**: レイアウト上の設備配置順序が工程フローチャートの工程順序と一致していることを検証する
3. **実物大テープ貼りで検証する**: 設備導入前に実物大のテープを床面に貼り、作業動線・保全スペースを実際に歩いて確認する
4. **3D CADシミュレーションを活用する**: 設備の干渉・搬送ルート・作業者の動線を3次元で検証する
5. **安全基準を設計初期に組み込む**: 消防法の避難距離・通路幅の基準を最初に確認し、制約条件としてレイアウトに反映する
6. **部品供給方式と連動して設計する**: ラインサイドの部品棚・AGVルート・フォークリフト通路を生産方式と合わせて計画する
7. **将来の増産・品種追加を見据える**: 初期計画で将来の拡張スペースを確保し、レイアウト変更の影響を最小化する
8. **現場の声を反映する**: 類似ラインの作業者・物流担当者・保全担当者のフィードバックを設計に反映する
